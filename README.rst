Goal
====

Help keeping Test Driven Development flow.

Installation
============

Install TddMon:

.. code-block:: console

   pip install tddmon

or current development version:

.. code-block:: console

   pip install hg+https:://bitbucket.org/kidosoft/tddmon

Usage
=====

.. code-block:: console

   tddmon -l test_run.log test_unit.py
